## 説明
sm******_ニコニコ動画でのタイトル.mp3 というネーミングがされている mp3 ファイルにファイル名からタグを設定するためのツール。
基本的には Vocaloid カテゴリで使われているパターンを元に正規表現のパターンマッチングでタイトルを抽出します。
個人的な管理方法にあわせてアルバム名にボカロP名を入れています。
ファイル名からだけでは決定できない時のためにニコニコ動画の該当ページから詳細説明を取得したり、投稿者名を取得する機能が
ついています。

mp3 をダウンロードする機能はありません。
